User Interface for staking NU tokens.

## Run project locally
1. Clone repository and install dependencies:
```
$ git clone https://github.com/ronmnm/stakeit.space.git
$ cd stakeit.space
$ npm install
```
2. Start the app on port 3000:
```
$ npm start
and navigate to http://localhost:3000/
```
